# 🧩 مصفوفة ميزات خادم LSP للغة ص

حالة المزوّدات ومصدر معرفتها. **مصدر**: من أين يستمدّ المزوّد معرفته (مصدر الحقيقة، نظام الأداة، أو منطق ساكن).

| المزوّد | الحالة | المصدر | ملاحظات |
|---|---|---|---|
| إكمال (Completion) | موصول | المصدر + نظام الأداة | إكمال طرق الأنواع بعد النقطة من `ALL_TYPE_METHODS` |
| تشخيصات (Diagnostics) | موصول | المصدر | دقّة كاملة بعد المرحلتين 2–3 |
| تحويم (Hover) | موصول | المصدر + نظام الأداة | أوصاف الكلمات من نظام الأداة؛ الأنواع/المدمجات من المصدر |
| تلوين دلاليّ (Semantic Tokens) | موصول | المصدر | أنواع سطحية + كلمات سياقية من المصدر؛ أطوال UTF-16 |
| تعريف (Definition) | موصول | فهرس الرموز | محلّيّ ← مستورد ← مدمج |
| مراجع (References) | موصول | فهرس الرموز | — |
| رموز المستند/المساحة (Symbols) | موصول | AST | — |
| تلميح التواقيع (Signature Help) | موصول | المصدر | تواقيع المدمجات من السجلّ |
| إعادة تسمية (Rename) | موصول | فهرس + حارس الكلمات | يمنع تسمية كلمةٍ محجوزة |
| إجراءات الكود (Code Actions) | موصول | منطق ساكن | — |
| تلميحات داخليّة (Inlay Hints) | موصول | تحليل الأنواع | يخفي نوع الإرجاع «فراغ» |
| طيّ (Folding) | موصول | AST | — |
| عدسات الكود (Code Lens) | موصول | AST | — |
| نطاق التحديد (Selection Range) | موصول | AST | — |
| شجرة الاستدعاءات (Call Hierarchy) | موصول | فهرس الرموز | — |
| شجرة الأنواع (Type Hierarchy) | موصول | تحليل الأنواع | الوراثة (آباء/أبناء) |
| روابط المستند (Document Links) | موصول | المحلّل | مسارات الاستيراد القابلة للنقر |
| التنسيق + أثناء الكتابة (Formatting) | موصول | sad_formatter | — |
| كاشف المشاكل المعروفة | موصول | منطق ساكن | يحذّر من أنماط موثَّقة المشاكل |
