## 🎯 الهدف

تمكين بناء تطبيقات واجهة **كاملة** بلغة ص مباشرةً — عناصر تصريحيّة (`واجهة ... نهاية`)، حالة تفاعليّة (`@حالة`)، ومعدّلات انسيابيّة (`.لون(...).عند_النقر(...)`) — فوق **مصدر بيانات موحَّد** (`language-truth/builtins/ui_widgets.yaml`)، مع **تكافؤ مفسّر↔مترجم** مُثبَت بالتشغيل لا بالوعد.

المستفيد: مطوّر التطبيقات العربيّ الذي يريد واجهةً أصليّةً دون مغادرة اللغة، وفريق اللغة الذي يريد محرّكين (تفسير + ترجمة أصليّة) بسلوك متطابق.
